# LaquaiLib.Analyzers

A collection of analyzers for C# projects.

## Packages

### [LaquaiLib.Analyzers](LaquaiLib.Analyzers/README.md)

Provides the analyzers.

### [LaquaiLib.Analyzers.Fixes](LaquaiLib.Analyzers.Fixes/README.md)

Provides code fixes for the analyzers in LaquaiLib.Analyzers.

### [LaquaiLib.Analyzers.Shared](LaquaiLib.Analyzers.Shared/README.md)

Internal helpers. Not indended for direct consumption.
